## Description

**TipMaster** est une application web interactive qui permet de calculer facilement un pourboire au restaurant. Elle permet de diviser l'addition équitablement entre plusieurs personnes avec une gestion intelligente des erreurs.

Ce projet a été réalisé dans le cadre d'un exercice de programmation front-end (JavaScript DOM, événements, validation de données).

##  Fonctionnalités

- Saisie du montant total de l'addition
- Choix du pourboire (5%, 10%, 15% ou valeur personnalisée)
- Nombre de personnes ajustable
- **Validation intelligente** : empêche les valeurs négatives, les champs vides et le nombre 0
- Formatage automatique à **2 chiffres après la virgule**
- Affichage en temps réel :
  - Montant du pourboire
  - Total général (addition + pourboire)
  - Total à payer par personne
- Bouton **Réinitialiser** pour remettre tous les champs à zéro
- Interface moderne, responsive et accessible sur mobile
